# Uレジ FOOD トレーニングシミュレーター

## 📱 概要

USENレジ（Uレジ FOOD）の操作習得を目的とした、トレーニング用Webアプリケーションです。
練習モードが存在しない本番機に対して、新人スタッフがリスクなく「開店〜注文〜会計〜トラブル対応〜閉店」の一連の業務フローを反復練習できる環境を提供します。

**コンセプト**: 「見た目はUSENレジ、機能は現場マニュアル準拠」

## ✨ 主な機能

### 📋 基本業務フロー
- **開店準備（レジオープン）**: 準備金入力（金種別）
- **注文入力**: テーブル選択、人数入力、商品選択、厨房送信
- **会計処理**: 現金/カード/QR決済対応、釣銭計算
- **閉店作業（レジクローズ）**: 現金有高点検、過不足確認、翌日準備金設定

### 🎯 応用機能
- **伝票分割（割り勘）**: 注文内容を分けて会計
- **テーブル合算**: 複数テーブルを1つの会計にまとめる
- **赤伝票（返品処理）**: 会計済み取引の返品対応
- **Wi-Fi切断シミュレーション**: ネットワークトラブル対応の練習

### 💡 トレーニング特化機能
- **ドロワー閉め忘れ防止**: 会計後に強制表示される警告画面
- **注文送信演出**: 厨房への送信完了トースト表示
- **過不足アラート**: 現金有高点検での差異検出

## 🚀 使い方

### 1. セットアップ

このプロジェクトは完全な静的サイトとして構築されており、特別なビルドやインストールは不要です。

```bash
# ファイルをダウンロードまたはクローン
git clone [repository-url]
cd USENRegister_iPad

# ローカルサーバーで起動（任意のHTTPサーバーを使用）
python -m http.server 8000
# または
npx serve
```

ブラウザで `http://localhost:8000` にアクセスしてください。

### 2. ログイン

デフォルトのログイン情報:
- **ID**: `admin`
- **パスワード**: `admin`

または、`data/staff.json` に登録されているスタッフIDでログイン可能です。

### 3. 開店準備

ログイン後、準備金入力モーダルが表示されます。
- 各金種の枚数を入力
- 合計金額を確認して「確定」

### 4. 業務操作

#### 注文入力
1. ダッシュボードから「オーダー会計」を選択
2. 空席テーブルをタップ → 人数を入力
3. カテゴリから商品を選択
4. 「確定」で厨房に送信

#### 会計
1. ダッシュボードから「会計」を選択
2. 会計対象のテーブルを選択
3. 支払方法（現金/カード/QR）を選択
4. 現金の場合、テンキーで預かり金額を入力
5. 「会計完了」→ ドロワーを閉める

#### 取引履歴・返品
1. ダッシュボードから「取引履歴」を選択
2. 返品したい取引の「返品」ボタンをタップ
3. 警告内容を確認して「返品実行」

#### 点検・精算
1. ダッシュボードから「点検・精算」を選択
2. 実際のドロワー内の現金を金種別に入力
3. 理論値と実査高の差異を確認
4. 翌日準備金額を設定
5. 「精算完了」

### 5. トラブルシミュレーション

#### Wi-Fi切断対応
1. ダッシュボードから「設定」を選択
2. 「Wi-Fi切断シミュレート」をクリック
3. 接続エラー状態を体験
4. IPアドレス確認手順を練習
5. 「Wi-Fi再接続」で復旧

## 🎨 UI/UXの特徴

### カラーテーマ
- **USENブルー** (`#0056b3`): ヘッダー、主要強調箇所
- **オレンジ** (`#f0ad4e`): 確定・会計ボタン
- **赤** (`#d9534f`): 取消・削除・返品ボタン
- **グリーン** (`#5cb85c`): 成功メッセージ

### 実機再現要素
- アナログ時計ウィジェット（ダッシュボード）
- 天気情報ウィジェット
- テーブルマップのステータス色分け
  - 空席: 白
  - 接客中: 青
  - 会計待ち: オレンジ

### 操作性
- タップ時のスケールアニメーション

## 🛠️ カスタマイズ方法

### 商品の追加・編集

`data/products.json` を編集してください:

```json
{
  "id": 35,
  "categoryId": 1,
  "name": "新メニュー",
  "price": 1000
}
```

### スタッフの追加

`data/staff.json` を編集してください:

```json
{
  "id": "004",
  "name": "田中次郎",
  "password": "9999",
  "role": "staff"
}
```

### カテゴリの変更

`index.html` の `initializeCategories()` メソッド内で編集可能です。

## 📱 推奨環境

- **デバイス**: iPad（横向き）、PCブラウザ
- **ブラウザ**: Chrome, Safari（最新版）
- **解像度**: 1024x768以上

### iPadでのフルスクリーン表示

1. Safariでアプリを開く
2. 共有ボタン → 「ホーム画面に追加」
3. ホーム画面のアイコンからアプリを起動

## ⚠️ 注意事項

- このアプリは**トレーニング専用**です。実際の会計には使用しないでください。
- データはブラウザのメモリ上に保持され、リロードすると全てリセットされます（意図的な仕様）。
- 本番機（USENレジ）とは一部機能が異なる場合があります。

## 🎓 トレーニングシナリオ例

### 初級: 基本オペレーション
1. 開店準備金10万円を入力
2. テーブル1に2名様を案内
3. 定食2品を注文入力
4. 現金で会計（1,000円札でお預かり）
5. ドロワーを閉める

### 中級: 複数テーブル対応
1. テーブル1〜3に順次お客様を案内
2. それぞれ異なる注文を入力
3. テーブル1から順に会計
4. カード決済とQR決済を練習

### 上級: トラブル対応
1. 会計済み取引の返品処理
2. Wi-Fi切断時の対応手順
3. 現金過不足が発生した場合の精算
4. テーブル合算機能の利用
